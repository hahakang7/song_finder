# song_finder
이 프로젝트는 유튜브뮤직에서 내가 좋아하는 가수를 팔로우했을때 그 가수의 신곡을 관리해주는 프로그램입니다

우리가 관심을 가지는 가수가 한둘이 아니기때문에 모두 모니터링 하는것도 불가능하다.

이미 알고있는 곡과 모르는곡이 섞여서 찾기도 쉽지 않다.
